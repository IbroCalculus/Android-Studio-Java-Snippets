NB: To work with FragmentList OR ListView within fragment, reference same as above.


=== CREATE THE FRAGMENTS =====
NB:   After creating a fragment, in the java class, delete all except as shown below:


public class ListFrag extends Fragment {

    public ListFrag() {
        // Required empty public constructor
    }


    @Override
    public View onCreateView(LayoutInflater inflater, ViewGroup container,
                             Bundle savedInstanceState) {
        // Inflate the layout for this fragment
        return inflater.inflate(R.layout.fragment_list, container, false);
    }
}



==== LINK FRAGMENT TO ACTIVITY ========
NB: Better to place fargments in activity via linear layout.
    set either layout_width = "0dp" or layout_height = "0dp" depending on orientation
    Set weight value. ie layout_weight = "1"
    If fragment does not appear in MainActivity Design view, use code <fragment ...

    <androidx.fragment.app.FragmentContainerView
        android:id="@+id/list_frag"
        android:layout_width="0dp"
        android:layout_height="match_parent"
        android:layout_weight="1"
        tools:layout="@layout/fragment_list" />
